🚀 立即使用
────────────────────────────────────────────────────────────

1. 啟動 Backend:
   cd /home/akira/github/9Jang/backend
   pnpm run start:dev

2. 訪問 Swagger UI:
   http://localhost:3000/api/docs

3. 測試 API:
   a) 點擊 POST /api/account/login
   b) 輸入帳號密碼登入
   c) 複製返回的 access token
   d) 點擊右上角 "Authorize" 按鈕
   e) 輸入 token 並確認
   f) 現在可以測試所有 API！
